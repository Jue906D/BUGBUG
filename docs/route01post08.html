二进制的祖宗：从《易经》和阴阳调和说起

二进制数系统，作为现代计算机科学的核心基础之一，由德国数学家戈特弗里德·威廉·莱布尼茨（Gottfried Wilhelm Leibniz）在17世纪首次系统地提出。

莱布尼茨在1703年发表的论文《二进制系统的理性思考》中介绍了二进制数系统。他认为，二进制不仅仅是一种数学工具，还与自然界的基本结构和哲学原理息息相关。

据史料记载，莱布尼茨曾因友人获得了《易经》六十四卦木刻图版。他注意到，这些卦象的结构与二进制数系统有相似之处：每个卦象由六条爻组成，每条爻有阴（断线）或阳（不断线）两种状态，恰好对应二进制的“0”和“1”。

换言之，他认为：古代中国“伏羲八卦”体系蕴含了与现代二进制同构的思路。

如果我们往回追溯《易经》，会发现一件很有意思的事——在《易经》八卦体系中，常见八个基本卦象（例如乾、坤、震、巽、坎、离、艮、兑）。这些八种状态，可按三爻（每爻断／不断）编码成 0 至 7 的数字，用二进制位 000 至 111 映射。

这样，八卦的“八”就与三位二进制数完全对应，比如巽卦“☴”顺时针旋转90°就是二进制011，换算成十进制也就是3。其后叠加两卦（六爻）即可映射 0 至 63（＝2⁶ − 1），与现代二进制系统的 6 位编码结构一致。

这一观点虽未必为古人刻意为之，但从后世哲学与数理视角看，却具“祖宗”意义。因此，当我们在使用计算机、敲击键盘、观察芯片电路的开关状态时，不妨回顾一下五千年前中华文明的瑰宝《易经》，以及它所蕴含的深邃哲学思想。
